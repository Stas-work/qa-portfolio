# Тест-кейсы: Логин
Сайт: https://www.saucedemo.com/

### TC-01: Успешный вход с валидными данными (standard_user)
Предусловия:
- Открыта страница логина

Шаги:
1. Ввести Имя пользователя: standard_user
2. Ввести Пароль: secret_sauce
3. Нажать Login

Ожидаемый результат:
- Пользователь авторизован, открыта страница товаров

### TC-02: Ошибка при пустых полях логина
Предусловия:
- Открыта страница логина

Шаги:
1. Не заполнять Имя пользователя
2. Не заполнять Пароль
3. Нажать Login

Ожидаемый результат:
- Авторизация не выполняется, отображается сообщение об ошибке, пользователь остаётся на странице логина
