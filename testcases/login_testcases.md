# Тест-кейсы: Логин
Сайт: https://www.saucedemo.com/

### TC-01: Успешный вход с валидными данными (standard_user)
Предусловия:
- Открыта страница логина

Шаги:
1. Ввести Имя пользователя: standard_user
2. Ввести Пароль: secret_sauce
3. Нажать Login

Ожидаемый результат:
- Пользователь авторизован, открыта страница товаров

### TC-02: Ошибка при пустых полях логина
Предусловия:
- Открыта страница логина

Шаги:

2. Не заполнять Пароль

### TC-10: Logout - back - нет доступа к товарам
Предусловия:
- Открыта страница логина

Шаги:
1. Ввести Имя пользователя: standard_user
2. Ввести Пароль: secret_sauce
3. Нажать Login
4. Открыть меню (кнопка ☰ слева сверху)
5. Нажать Logout
6. Нажать Back в браузере

Ожидаемый результат:
- Пользователь не попадает на страницу с товарами без авторизации, видит страницу логина
- При попытке вернуться назад может отображаться сообщение о необходимости авторизации
