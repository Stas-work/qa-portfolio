# При отправке формы с пустыми полями Имя пользователя и Пароль отображается ошибка только для поля Имя пользователя, нет явной подсказки по полю Пароль 

Предусловия:
Открыта страница логина https://www.saucedemo.com/

Шаги:
1. Не заполнять Имя пользователя
2. Не заполнять Пароль
3. Нажать Login
   
Ожидаемый результат:
Под полями появляется сообщения об ошибке в Имя пользователя и Пароль

Фактический результат:
Под полями отображается сообщение об ошибке: Epic sadface: требуется имя пользователя, про поле Пароль подсказки нет, при этом поле отмечено красным как обязательное 

Окружение:
Windows 10 Pro, Yandex Browser 25.10.2.1176 (64-bit)

Важность:
Низкая

Приоретет: 
Низкий
