# Появляется сообщение об ошибке в Имя пользователя и Пароль при пустых полях

Предусловия:
Открыта страница логина

Шаги:
1. Не заполнять Имя пользователя
2. Не заполнять Пароль
3. Нажать Login
   
Ожидаемый результат:
Под полями нет сообщения об ошибке

Фактический результат:
Под полями отображается сообщение об ошибке: Epic sadface: требуется имя пользователя, пользователь остаётся на странице логина 

Окружение:
Windows 10 Pro, Яндекс браузер версия: 25.10.2.1176 (64-bit)
