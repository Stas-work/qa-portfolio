# При пустых полях Имя пользователя и Пароль отображается ошибка только для поля Имя пользователя

Предусловия:
Открыта страница логина https://www.saucedemo.com/

Шаги:
1. Не заполнять Имя пользователя
2. Не заполнять Пароль
3. Нажать Login
   
Ожидаемый результат:
Отображается сообщение об ошибке, что оба поля обязательны (Имя пользователя и Пароль)

Фактический результат:
- Отображается сообщение об ошибке: Epic sadface: требуется Имя пользователя
- Для поле Пароль подсказки нет (только красная подцветка) 

Окружение:
Windows 10 Pro, Yandex Browser 25.10.2.1176 (64-bit)

Важность:
Незначительно

Приоретет: 
Низкий
